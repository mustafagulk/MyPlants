✏️ How to Edit This File

The file is a single HTML file — you edit it in any text editor

You don't need any special software. Notepad on Windows or TextEdit on Mac works fine. For a better experience, use VS Code (free, download at code.visualstudio.com) which highlights the code in colour and makes it much easier to read.

  1. Right-click the HTML file → Open with → Notepad (or VS Code)
  2. Press Ctrl+F (Windows) or Cmd+F (Mac) to search
  3. Search for the plant name you want to change, e.g. tomato or Strawberry
  4. Find the value you want to edit and change the number
  5. Save the file (Ctrl+S), then refresh the browser tab
🔍 Where to find each type of data
Plant Advisor nutrient values (N, P, K, Ca, Mg, EC, pH, notes, fertilizer doses)
Search for const NUTRIENT_DATA — this is the main data block.
Fertilizer dose tables (the ppm numbers in Bio Grow / Bio Bloom / Primaster tables)
Search for Bio Grow or Bio Bloom — you'll find the table rows.
Adding a new plant to the Advisor
Search for const PLANTS — add a new entry, then match it in NUTRIENT_DATA.
